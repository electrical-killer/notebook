实现一个根据外部输入, 而动态输出的需求

- 无输入回车, 输出默认内容
- 输入内容, 动态输出

## 实现

```bat
@echo off

set /p input=Please enter a value(Vx.x.x.xxxxxx.xxxx): 

if "%input%" == "" (
	set input=Vx.x.x.000000.0000
) 

echo %input%

pause exit 
```
