## 来源

- 参考文章[起始页home3.0](https://wiki.onedayxyy.cn/docs/home3.0)
- demo仓库[onlyonexl/home3.0](https://gitee.com/onlyonexl/home3.0)
    - 该仓基于原作者[南淮花酱](https://github.com/hsBUPT/hsBUPT.github.io)仓库, 修复背景不能切换问题

## 亮点

- 风格亮眼, 还有看板娘..
- 鼠标指向头像, 可翻转
- 有搜索框
- 介绍信息栏, 点击后可展开第二屏的功能区
    - 时间胶囊
    - 网站链接
- 第二屏默认功能展示区
    - 一言&音乐播放器
        - 可以互相切换
        - 可以配置网站壁纸切换方式
    - 时间展示栏
    - 网站列表
- ……………
