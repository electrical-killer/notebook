## typora

#### 1. nginx上传新资源后, typora需要重启才能加载显示新的图片, 是否有优化空间?

- 尝试端口映射的时候, 将之前的 `域名+局域IP` 的方式, 更改为 `域名+公网IP` 方式, 莫名奇妙访问异常快速
- 且==内网访问==, sftp服务, nginx服务均正常
